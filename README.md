# Venv
a virtual enviroment to make programing cross platform desktop applications easyer.


# Features:

    1): LuaJIT
    
    2): Go
    
    3): Zig * Linux
    
    5): C /* zig cc */ * Linux
    
    6): C++ /* zig cc */ * Linux
